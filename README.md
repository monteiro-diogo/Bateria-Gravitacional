## 🔋 Sistema de Monotorização - Bateria Gravitacional 
Este repositório contém o sistema de gestão de energia desenvolvido para a unidade de Projeto 2 da Licenciatura em Engenharia Eletrotécnica e de Computadores. O projeto foca-se na monitorização e ativação automática de baterias de reserva para garantir a continuidade da produção elétrica em situações de falha na rede pública.

### 📁 Conteúdo
`/data`: Base para o site.  
`/include`: Headers "sem par".  
`/src`: Código fonte em C++.  
`/lib`: Bibliotecas privadas.  
`/test`: Para testes.
